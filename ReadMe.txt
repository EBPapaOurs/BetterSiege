Construction Mode :

	Clic gauche pour placer/détruire (changer direction des roues si clic sur une roue)
	Clic droit pour tourner la caméra
	Molette pour zoomer
	Clic molette pour reset la caméra
	Barre Espace pour passer dans le mode Déplacement

Déplacement Mode :

	Flèches haut/bas pour avancer/reculer
	Flèches gauche/droite pour tourner
	Clic droit pour tourner la caméra
	Molette pour zoomer
	Clic molette pour reset la caméra (bug parfois)

Canvas :

	Boutons du bas pour changer le cube à poser
	Bouton Destruction pour supprimer des blocs
	Bouton Reset pour relancer la scène courante
	Bouton Test pour lancer la scène de Test
